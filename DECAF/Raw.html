Raw Data


The Raw data (about 60 hours of MEG recordings at the sampling rate of 1KHz) has a size of about 300GB and sharing it on the web is not straight-forward. However, we have some solutions.


Please Read Me!

Please make sure that you consider the following hints before submitting your access-request:

- The data is shared only for academic research.
- Commercial-related use of the data is not permitted.
- If you use any part of the shared data in any report, please make sure that you cite the "DECAF Dataset paper" in the report.

Thanks for your consideration!


Access Request

We have certain tools to ship you the dataset. For further information, please contact us at decaf.mhug [at] gmail [dot] com .
